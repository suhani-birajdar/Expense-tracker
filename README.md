# Expense Tracker

A responsive expense tracker built with React.js featuring real-time calculations, category filters, and LocalStorage persistence.

## Features
- Add, edit, and delete expenses
- Category-wise breakdown (Food, Travel, Health, Grocery, etc.)
- Interactive charts with Chart.js
- Search and filter expenses
- Data saved in LocalStorage (persists on refresh)
- Responsive sidebar layout

## Tech Stack
- React (Vite)
- Chart.js
- CSS Flexbox

## How to Run
npm install
npm run dev
